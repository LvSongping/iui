iui的布局采用栅格思想进行设计和实现，并且支持除IE6-之外所有浏览器。iui布局选定的槽宽和槽间距都为1个字符的宽度，主要特性有：
  1. 可计算性，宽度=字符个数
    Example：
    iui布局
    w6指定的宽度就为6个字符的宽度，若字体为12px，则宽度就为72px。这样UI设计器只需了解需放多少个字符即可，无需象以往问一些例如多少像素的专业性问题。
  2. em单位，这样便可通过改变默认字体大小自动改变可个DOM相应长度和宽度。
  3. 弹性布局，iui布局除提供固定宽度样式库之外，还提供丰富百分比宽度定义，使DOM可随着不同场景自适应宽度
